<h3>install</h3>
 - hit KDE's Discover or Gnome's Software

 - search: shazware

 - click install

 - now they're in your start menu's <b>development</b> menu


<h3>summary</h3>
 - `bg2`fp2sh` - flatpak commands in CLI without the long dumb suffixes

 - `bg3`big` - print out text in a big font (like in the mainframe days)

 - `bg4`flatten` - flatten all the subdirs in a dir renaming files
   to dir1_dir2_file

 - `bg5`ftx` - find a text string in all the files of a dir tree

 - `bg6`l` - list files (i know i know but i hate ls)

 - `bg7`recent` - list files most recently touched in a dir tree

 - `bg8`renem` - rename all the files in a dir at once with a text editor

 - `bg9`undup` - kill off exactly duplicate files in a dir tree
   regardless of filename

 - `bga`ned` - `bga`n`ifty text `bga`ed`itor.
   I write all my code in it, but it's limited to 80 cols
